# 虚拟电厂交易与调度决策辅助系统

这是一套面向虚拟电厂的电力交易求职演示项目，用模拟数据展示分布式光伏、储能、工商业可调负荷和充电桩的聚合、预测、日前交易、协同调度、收益回测和辅助服务能力。

## 主要内容

- `outputs/虚拟电厂交易与调度决策辅助系统.html`：可直接打开的中文演示页面
- `outputs/新能源电力交易决策辅助系统演示.pptx`：面试演示 PPT
- `outputs/新能源电力交易决策辅助系统PRD.md`：中文 PRD
- `outputs/三分钟面试讲稿.md`：面试讲稿
- `outputs/面试追问答辩稿.md`：追问答辩
- `src/`：模拟数据、预测、交易策略、虚拟电厂调度、回测、储能模拟和报告生成代码
- `tests/`：核心逻辑测试

## 重要说明

本项目使用模拟数据演示虚拟电厂交易与调度决策流程，不代表真实交易收益，不构成交易建议。系统定位是辅助决策，不做实盘自动下单。

## 本地运行

使用 Codex 自带 Python 或本机 Python 运行：

```bash
python app.py
```

运行后会生成：

```text
outputs/虚拟电厂交易与调度决策辅助系统.html
```

## 测试

```bash
python -m unittest discover -s tests -v
```
